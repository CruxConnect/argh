Dependencies
------------

The `argh` library is supported (and tested unless otherwise specified) on
the following versions of Python:

* 2.6 (`argparse` library is required)
* 2.7 (including PyPy 1.8)
* 3.1 (`argparse` library is required; **not** tested)
* 3.2
* 3.3
